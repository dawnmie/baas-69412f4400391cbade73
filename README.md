# Appwrite + React Demo

This is a simple demo project using Appwrite and React.

## Setup

```bash
npm install
npm run dev
```

## Configuration

The Appwrite client is configured in `src/appwrite.js`:

- **Endpoint**: https://appbuild.store/v1
- **Project ID**: 69412f4400391cbade73

## Features

- User registration
- User login
- User logout
